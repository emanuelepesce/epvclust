The folder cointains:

- epvclust_C: (directory), cointains the code in C for doing hierarchical
			  clustering and calculating frequencies.
			  
- epvclust: (directory), epvclust complete package. It's an R package that 
			needs to be installed

- exampleData: (directory), cointains some example of dataset for doing testing

- install.R: file for installing epvclust package in R.
